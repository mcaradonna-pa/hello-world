UCD Migration for FreePrints Cards

Current FC Designer

  1. The cursor is in the preview area at the top of the page. Keyboard entries are displayed directly in the preview.
  2. Updates are done in client with no calls to the server and no delays. The keyboard is always displayed in the text edit mode.
  3. The Done button lives at the top right of the page.

UCD FC Designer

  1. In UCD, the cursor displays in a text box. Tapping into that text box makes the keyboard display.
  2. Updates are sent to servers and then back to client where they are updated in the preview area.
  3. The Done is still up at the top of the page. Tapping it drops the keyboard, leaving a large empty area.

Simply To Impress UCD

UCD is currently used in the STA app.

  1. There is no big blank space because all of the text edit tools live there.
  2. The Done button is closer to the text box where the user's eyes are, no Done button at the top.
  3. STA is on live servers. This is the expectation for text update speed when FC is running on live servers.

If you want to test UCD on FreePrints Cards yourself you can follow these steps.

  1. Open the App Distribution app.
  2. Select FreePrints Cards
  3. Select the latest version of 70.16.0 at the top.
  4. Open the app
  5. Tap the Main Menu button in the upper-left corner of the home page.
  6. Tap Help
  7. Tap FAQs
  8. Rapidly tap the FC logo at the top of the page 15 times.
  9. When the dialog displays enter "Hint Tft". Make sure there are no trailing spaces and mind the capitalization.
  10. Tap Stage on the Debug Menu dialog.
  11. Select cards-tran-ucd(PRE EU). The app will restart.
  12. Select any card.
